#  NumberNinja

NumberNinja is a simple and interactive **Guess the Number** game built using **HTML, CSS, and JavaScript**.  
The player has to guess a randomly generated number between **1 and 100**, with instant feedback after each guess.

---

##  Features

- Random number generation between 1 and 100  
- User input validation  
- Feedback for high / low guesses  
- Attempt counter  
- Restart game functionality  
- Clean and responsive UI  

---

##  Tech Stack

- **HTML** – Structure of the game  
- **CSS** – Styling and layout  
- **JavaScript** – Game logic and interactivity
